수정사항을 작성한 후에 업로드 해보았습니다. 노마드 코더(콜롬비아 출신 한국인) 사이트에 가입+깃허브 연동+ python으로 웹스크래퍼 만들기 수강하였습니다. 유튜브를 통해서 간단하게 파이선 웹 스크래퍼 기능을 구현해보는것을 따라해보기만 하였는데 이것으로는 조금 부족하다고 느꼈기 때문입니다. 또한 국민취업지원제도 선수학습에 대한 조언 (유튜브 : 스칼라의 IT직업충전소)에 따라서 java와 html, 웹 스크롤러 등등 선수학습이 필요하다고 하였기 때문입니다. 비주얼스튜디오 extentions 에서 live server를 설치하고 우측 하단에 go live를 클릭하면 이제 저장만 하면 실시간으로 반영이 되는 웹 사이트를 만들었다. 주소는 http://127.0.0.1:5500/index.html5 이제는 extentions에서 IP Address를 설치해서 휴대폰에 아이피를 적었는데 안된다. 아마도 데스크탑과 휴대폰으로 실습을 해서 안되는것으로 추정된다. 동일한 와이파이에 접속한 상태 즉 노트북과 휴대폰과는 성공할 것 같다. 이고잉 web1 수업 마지막에 동영상 삽입과 댓글 기능 추가, 채팅 기능 추가, 방문자 분석기 추가 내용까지 포함시켰다. 비쥬얼스튜디오 : 웹 하나하나 저장을 반드시 해야 적용이 된다는것을 알게되었다. 쉬프트와 알트를 이용해서 한꺼번에 수정이 가능하다는것에 놀랐다. 컨트롤z를 누르면 단이 바뀌는것이 신기했다. 컨트롤v로 붙여넣은다음에 컨트롤z를 누르면 단의 위치가 바뀌는것이 신기했다. web1(html),2(css)로 비주얼 스튜디오와 깃허브로 웹 어플리케이션 제작(홈페이지 제작, 웹사이트 제작)을 하고, 반응형 웹도 만들어보고 좋은 경험이다. 그런데 단문 주석을 작성하는 방법은 잘 모르겠고, 점을 찍고 엔터를 치면 div class가 생성된다. 스페이스바를 치고 엔터를 쳐야된다. 내일은 학원 시작 전날이다. 그동안(약 한달)간에 준비했던것을 정리하고, 훈련기간동안의 목표를 수립하고, 마음의 준비를 하도록하자. 깃허브에 어수선하지만 정리해서 업로드하도록하자
